<h1 align="center"> Lists </h1>

```python
x = ["Khaled","Sayed","Noor", "Ahmed"]

type(x)     # list

x[1]        # sayed

x[0:2]      # ["Khaled","Sayed","Noor"]

x[1:]       # ["Sayed","Noor","Ahmed"]

len(x)      # 4

x.count("khaled")     # 1
```

### reverse() : Reverse the list

```python
x = ["Khaled","Sayed","Noor", "Ahmed"]

x.reverse()

print(x)      # ["Ahmed","Noor","Sayed","Khaled"]

```
